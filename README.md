# test-repo-mcp

这是一个通过 GitHub MCP 创建的测试仓库，用于验证 GitHub MCP 的各种功能。

## 项目描述

这个项目包含：
- Python 示例代码
- 单元测试
- 配置文件
- Git 分支管理示例

## 功能特性

- ✅ 基本的 Python 程序
- ✅ 单元测试框架
- ✅ 配置文件管理
- ✅ Git 工作流测试

## 使用方法

```bash
# 运行主程序
python3 main.py

# 运行测试
python3 test_main.py
```

```
test-repo-mcp/
```
